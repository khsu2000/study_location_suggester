Resources

ipdata.co: https://ipdata.co/
Ipdata.co is used to get the location of the user when he or she first visits the webpage. Using ipdata.co, the web app can get information about the user's location like the user's current city or lat/lon coordinates.
Foursquare: https://foursquare.com/
Foursquare is used to query nearby venues. Given search keywords, a search radius, and information about the user's location, the web app uses Foursquare's explore endpoint to get a list of relevant venues.
